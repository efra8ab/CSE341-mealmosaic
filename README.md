# Meal Mosaic API

Recipes CRUD with Express, MongoDB, and Swagger docs at `/api-docs`.

## Quickstart
1. `npm install`
2. Generate docs: `npm run swagger`
3. Run dev server: `npm run dev`. Swagger UI: `http://localhost:8080/api-docs`

## Delivered in first commit
- **Recipes**: GET all/one (public), POST/PUT/DELETE. Validates required fields, ingredient names, steps, and positive times/servings.
